[ OVERVIEW ]

Dark Mode Ultra goes beyond dark mode. Regular dark mode changes your background to dark gray like a coward. Dark Mode Ultra changes your background to absolute void. We're talking #000000. Pure black. The absence of all light. The color of space. The color of my soul.

But we didn't stop there. Dark Mode Ultra also darkens your text, your images, your scrollbar, your cursor, and eventually your monitor. If you can still see anything, Dark Mode Ultra has not finished its job.

[ INSTALLATION ]
import { enableDarkModeUltra } from 'bleep-dark-ultra'; enableDarkModeUltra({ level: 'extreme', // 'mild' | 'extreme' | 'void' | 'event-horizon' darkenImages: true, // makes all images darker. why? because. darkenText: true, // makes text slightly invisible. for aesthetics. darkenScrollbar: true, // the scrollbar is now a suggestion darkenCursor: true, // good luck finding it darkenDeveloper: false, // coming in v2. will dim your screen brightness. });
