An Open Letter to CSS

>> a breakup 10 years in the making
[ DEAR CSS, ]

We need to talk.

I know we've been through a lot together. The late nights. The tears. That time I spent four hours trying to center a div and ended up questioning whether I chose the right career. The way you gaslit me into believing margin: 0 auto would fix everything, when we both knew it wouldn't.

I thought we could make it work. I really did. But I've been seeing Tailwind behind your back, and honestly? It actually does what I ask it to.

This letter is my formal goodbye. My therapist said I need "closure." So here we are.

[ WHAT WENT WRONG ]

1. The Centering Incident. You know what you did. I said "center this vertically." You said "try vertical-align: middle." That was a lie, CSS. A cold, calculated lie. I tried display: table-cell. I tried line-height hacks. I tried transform: translateY(-50%). I ended up with position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%) like some kind of incantation from a forbidden grimoire. Flexbox eventually showed up and did it in two lines. Two. Lines.

2. The z-index Wars. I set something to z-index: 10. It didn't appear on top. I set it to 999. Still nothing. I set it to 2147483647. A dropdown menu from a WordPress plugin from 2014 was still on top of it. I have stared into the stacking context abyss, CSS, and the abyss had a tooltip that wouldn't go away.

3. Specificity Nightmares. I wrote a simple rule: .button { color: blue; }. It didn't work. Because somewhere, six files deep in a dependency I didn't install, someone wrote div.container > section.main article.content .wrapper .inner .button-group .button { color: red; } and now I need !important which, as we both know, is just a fancy way of admitting defeat.

4. Float: Left. I don't want to talk about it. We both know what happened. The clearfix hack was a war crime.

[ WHAT I'LL MISS ]

Let me think about this carefully.

[ CLOSING REMARKS ]

CSS, I want you to know something. I'm not angry. I'm disappointed. Which, as we both know, is worse.

If you ever show up at my door again with another "exciting new feature" like container queries or :has() selectors, I will call the police. You had decades to be good. You chose chaos.

I'm taking the kids (JavaScript and HTML). Don't contact my lawyer (Stack Overflow).

Goodbye forever,
